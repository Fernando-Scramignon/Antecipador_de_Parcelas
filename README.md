# Antecipador_de_Parcelas
Calculadora de antecipação

## deploy link
[Antecipador de Parcelas](https://antecipador-de-parcelas.vercel.app)

## Como rodar localmente

- Clone o repositório
- Instale as dependências
  - ex: ```yarn``` 
- rode o comando ```dev```
  - ex: ```npm run dev```
  - ex: ```yarn dev```
